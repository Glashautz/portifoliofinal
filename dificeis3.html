PA e PG

ambas são sequencias formadas por um termo incial e uma razão podendo ser finitas ou infinitas. Podendo ser de numeros naturais, inteiros racionais etc.

Progressão Aritimetica

PA são sequencias onde a razão é somada ao termo, ou seja pode ser representado pela formula an+r. Ou seja>

  • a1 + r = a2
  • a2 + r = a3
  • a3 + r = a4

Para descobrir a razao é necessario subtrair dois valores um perto do outro:

r = an - an-1

Classificação

  • Cresecente: quando r > 0
  • Constante: quando r = 0
  • decrescente: quando r < 0

Termo geral

Formula utilizada para descobrir qualquer elemento dentro de uma PA

an = a1 + (n -1)r

Interpolação aritmética

Nas sequências finitas (𝑎1, 𝑎2, … , 𝑎𝑛) os termos 𝑎1 𝑒 𝑎𝑛 são chamados extremos e os demais são chamados meios. 7 Interpolar, intercalar, inserir 𝑘 meios aritméticos entre dois números 𝒂 e 𝒃, significa obter uma PA de extremos a e b com n = k + 2 termos. Para encontrar esses termos temos que encontrar a razão.

Soma de termos PA finita

A formula para somar todos os termos de uma PA finita é:

Sn = (a1 + an)n / 2

Progressão Geométrica

PG são sequencias onde a razão é multiplicada ao termo, ou seja pode ser representado pela formula an * r. Ou seja>

  • a1 * r = a2
  • a2 * r = a3
  • a3 * r = a4

    • Para descobrir a razao é necessario dividir dois valores um perto do outro:

    r = an / an-1

    PG Cresecente:

    • PG com termos positivos: r > 1.
    • PG com termos negativos: 0 < r < 1.

    PG Constante:

    • PG com todos os termos nulos: 𝑎1 = 0 𝑒 𝑞 𝑞𝑢𝑎𝑙𝑞𝑢𝑒𝑟.
    • PG com termos iguais e não nulos: 𝑎 ≠ 0 𝑒 r = 1.

    PG Decrescente:

    • PG com termos positivos: 0 < r < 1
    • PG com termos negativos: r > 1
    • Alternantes: cada termo tem sinal contrário ao do termo anterior. Isso ocorre quando r < 0.
    • Estacionárias: Apenas o primeiro termo é diferente de zero: 𝑎1 ≠ 0 𝑒 r = 0.

    Termo geral

    an = a1 ∙ rn-1

    Soma de termos de PG finita

    Sn = a1 * 1-rn / 1-r

    Soma dos termos de uma PG infinita, quando -1 < r < 1.
